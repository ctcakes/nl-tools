# nl-tools
## NL注入辅助工具
# 为什么？
## 由于勾栏子V🐍更新64位Steam，导致NL注入器拉起Steam后Steam闪退。为解决这个问题，写了这个小工具。
# 怎么工作？
## 通过读取%temp%下的NL注入器日志，当检测到注入器初始化完成并且拉起Steam时，将注入器拉起的Steam结束进程（不结束也会自己闪退），然后通过本工具拉起游戏。
## 游戏启动后NL便会进行注入流程。
# 附
## 本工具还会读取NL_CSGO的日志，在选择CSGO游戏后，NL注入器初始化完成就会开始监听，实时反馈注入进度（服务器连接成功、用户信息获取成功、开始注入、注入完成）
# ⚠️
## 使用本工具不会造成你的NL被ban，本工具完全使用文件系统进行读取注入器日志实现所有功能，无需担心风险问题。
